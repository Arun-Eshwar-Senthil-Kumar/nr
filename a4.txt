🔧 PART 1: Install & Run Basic Jenkins Job
✅ 1. Install Requirements
Install Java
Install Jenkins
✅ 2. Start Jenkins

Open browser:

http://localhost:8080
Unlock using password
Install Suggested Plugins
✅ 3. Create First Job
Click New Item
Name: hello-job
Select Freestyle Project
✅ 4. Add Build Step
Go to Build → Add build step
Select Execute Windows batch command

Paste:

echo Hello Jenkins
✅ 5. Run Job
Click Build Now
✅ 6. Verify
Click build → Console Output
✔ Must show:
Hello Jenkins

🔧 PART 4: Pipeline Job
✅ 1. Create Pipeline
New Item → pipeline-job
Select Pipeline
✅ 2. Add Script
pipeline {
    agent any

    stages {
        stage('Build') {
            steps { echo 'Build' }
        }
        stage('Test') {
            steps { echo 'Test' }
        }
        stage('Deploy') {
            steps { echo 'Deploy' }
        }
    }
}
✅ 3. Run
Click Build Now

✔ Check logs
